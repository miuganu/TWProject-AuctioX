Abstract

AuctioX este o aplicatie open source (https://github.com/miuganu/TWProject-AuctioX ) ce se doreste a fi in spiritul unui site specializat, si anume eBay. Aceasta aplicatie a fost conceputa in scop pur didactic reprezentand un proiect in cadrul materiei Tehnologii Web.

Introducere

AuctioX este o aplicatie web de realizare a licitatiilor de produse cu caracter personal, astfel o accesare a aplicatiei propriu-zise necesita logarea utilizatorului. Implicit in cadrul aplicatiei un utilizator(logat) va avea posibilitatea de a licita pentru produsele adaugate de alti utilizatori, dar si de a adauga produsele personale.

Fluxul de stiri vine ca si o functionalitate a aplicatiei punand la dispozitie informatii in timp real relativ la licitatiile noi si data limita pentru licitarea acestora.

La nivel inalt scopul acestei aplicatii este de a:

  • pune la dispozitie utilizatorului o gama cat mai larga de produse
  • reprezenta o platforma unde utilizatorii isi pot pune la dispozitie produsele pe care doresc sa le dea spre licitatie
  • facilita cautarea prin punerea la dispozitie a filtrelor(dupa categoria din care fac parte licitatiile, data de expirarea, si pretul maxim al produselor)
  • oferi posibilitatea de a cauta dupa cuvinte cheie prezente in titlul sau descrierea licitatiei

Structura

Pagina de pornire
• Va contine un formular de login unde sunt prezente 2 campuri si anume username si parola, o nepotrivire a acestora va genera diverse mesaje de eroare. De asemeni va exista un buton de Login care permite accesul catre pagina principala a aplicatiei. Un buton de Make an account va fi situat in partea dreapta a paginii de pornire, si va oferi utilizatorului posibilitatea de a crea un cont nou.

Pagina de inregistrare utilizator
• Cerintele pentru inregistrare sunt urmatoarele:
• Un username
• O parola (de minim 6 caractere)
• Confirmarea parolei
• In final un email(valid)

Pagina principala va contine
• Header-ul care contine numele aplicatiei(un simplu click va redirectiona utilizatorul catre pagina principala), un search bar prevazut cu un buton de search, apoi 2 dropdown-uri unul care pune la dispozitie formatele in care poate fi downloada situatia licitatiilor, si unul de options, care pune la dispozitie pagnile de MyAuction, AddAuction, News si Change password. Pe lanaga acestea se afla si butonul de logout.
• Licitatiile, fiecare licitatie va avea un titlu, descriere, pretul de start, pretul actual in cazul licitatiilor care nu au expirat(la aceste produse se poate licita), altfel va fi afisat pretul de vindere(in cazul produselor expirate). De asemeni sunt incluse si dropdown-ur care pun la dispozitie filtrarile disponibile, si anume dupa categorie, pret si data de expirare.

Pagina Add Auction
• Pagina de unde utilizatorul va putea adauga obiecte.
• Este necesara completarea tuturor campurilor pentru a adauga o licitatie si anume: numele produsului, pretul initial, categoria din care face parte, o imagine in diferite formate, o data de expirare care nu depaseste o luna de zile, si o descriere a produsului.
• Aceasta pagina este prevazuta si cu un buton care permite salvarea licitatiei adaugate.

Pagina MyAuction
• Reprezinta pagina cu licitatiile adaugate de catre utilizatorul existent in acea sesiune.
• Formatul in care acestea vor fi afisate este similar paginii principale.

Pagina de schimbare a parolei
• Este un form unde e necesara completarea a 3 campuri si anume: parola veche, parola noua si confirmarea acesteia.
• Contine un buton pentru salvarea noii parole.

Pagina de News
• Pune la dispozitie un flux de stiri, in format Atom, cu ultimele produse adaugate.

Utilizatorul înregistrat

Se logheaza pe aplicatie folosind un username si o parola. Dupa ce s-a logat are urmatoarele drepturi:
- adauga un nou prods spre a fi licitat;
- poate sa liciteze( cu un pret mai mare decat cel curent) la orice licitatie care nu e expirata;
- are acces la situatia licitatiilor;
- isi poate schimba parola (doar daca o cunoaste pe cea existenta);
- genereaza rapoarte despre anunturi in urmatoarele formate:JSON, PDF si XML;
- are acces la fluxul de stiri disponibil in format Atom.

Utilizator fara cont

Utilizatorul fara cont nu va putea accesa aplicatia. Acesta poate sa isi creeze un cont completand formularul de inregistrare.

Tehnologii utilizate

Proiectul utilizeaza urmatoarele tehnologii: HTML5 - este un limbaj pentru structurarea si prezentarea continutului pentru World Wide Web, o tehnologie nucleu pentru Internet propusa initial pentru software-ul Opera.

CSS - (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atasa elementelor HTML prin intermediul unor fisiere externe sau în cadrul documentului, prin elementul "style" si/sau atributul style. CSS se poate utiliza si pentru formatarea elementelor XHTML, XML si SVGL

Javascript - este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor.Este folosit mai ales pentru introducerea unor functionalitati în paginile web, codul Javascript din aceste pagini fiind rulat de catre browser. Limbajul este binecunoscut pentru folosirea sa in construirea siturilor web, dar este folosit si pentru acesul la obiecte incastrate (embedded objects) în alte aplicații. A fost dezvoltat initial de către Brendan Eich de la Netscape Communications Corporation sub numele de Mocha, apoi LiveScript, și denumit în final JavaScript.

PHP - Hypertext Preprocessor. Folosit initial pentru a produce pagini web dinamice, este folosit pe scară largă în dezvoltarea paginilor si aplicațiilor web. Se folosește in principal inglobat în codul HTML, dar incepand de la versiunea 4.3.0 se poate folosi și în mod „linie de comandă” (CLI), permițând crearea de aplicații independente. Este unul din cele mai importante limbaje de programare web open-source și server-side, existând versiuni disponibile pentru majoritatea web serverelor și pentru toate sistemele de operare. Conform statisticilor este instalat pe 20 de milioane de situri web și pe 1 milion de servere web. Este disponibil sub Licenṭa PHP ṣi Free Software Foundation îl consideră a fi un software liber.
